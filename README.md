# phymicro
# 🎓 Student Management System (Python)

## 📌 Overview

This is a simple **Student Management System** built using Python.
It allows users to manage student records through a menu-driven interface.

---

## 🚀 Features

* ➕ Add new student
* ✏️ Update student details
* ❌ Delete student
* 📋 Display all students
* 🔁 Menu-driven system for easy interaction

---

## 🛠️ Technologies Used

* Python 3
* Command Line Interface (CLI)

---

## ▶️ How to Run

1. Make sure Python is installed
2. Open terminal in project folder
3. Run:

```
python class2.py
```

---

## 📸 Sample Menu

```
1. Add Student
2. View Students
3. Update Student
4. Delete Student
5. Display Students
6. Exit
```

---

## 🧠 How It Works

* The program runs in a loop (`while True`)
* User selects options from menu
* Based on input:

  * Calls functions like `add_student()`, `update_student()`, etc.
* Exits when user selects option `6`

---

## ⚠️ Notes

* Data is stored temporarily (no database)
* Restarting program resets all data
